[![ROC-AUC](https://img.shields.io/badge/ROC--AUC-1.0-brightgreen?style=flat)](https://github.com/harshityadav802/Phishing-URL-Detector)

`Python` `CatBoost` `scikit-learn` `Pandas` `React` `REST API` `Vercel`

### Pneumonia Detection from Chest X-rays
> *Jan 2025 – Aug 2025 · [Code](https://github.com/harshityadav802/pneumonia-detection-project)*

[![Model](https://img.shields.io/badge/Model-ResNet18-purple?style=flat)](https://github.com/harshityadav802/pneumonia-detection-project)
[![XAI](https://img.shields.io/badge/Explainability-Grad--CAM-orange?style=flat)](https://github.com/harshityadav802/pneumonia-detection-project)
[![Dataset](https://img.shields.io/badge/Dataset-5%2C863%20X--rays-blue?style=flat)](https://github.com/harshityadav802/pneumonia-detection-project)

- Fine-tuned **ResNet18** on 5,863 chest X-ray images using PyTorch with targeted data augmentation and regularisation for class imbalance
- Integrated **Grad-CAM** (Gradient-weighted Class Activation Maps) for explainability — clinical heatmaps highlighting pathological lung regions
- Deployed **interactive Gradio app** for real-time X-ray upload, classification, and visual explanation

`Python` `PyTorch` `ResNet18` `Grad-CAM` `Gradio` `OpenCV` `Explainable AI`
